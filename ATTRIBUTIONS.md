# Attributions

This file tracks third-party assets used in Calmodoro so organizational deployments can verify licensing and attribution requirements.

## Libraries

1. **lottie-web**
   - Source: https://github.com/airbnb/lottie-web
   - License: MIT
   - Usage: Bundled player at `lottie/lottie.min.js`

## Animation assets

Add every third-party animation used in `lottie/animations/` here with:

- Asset file name
- Author/creator name
- Source URL
- License / free-use terms
- Any required attribution text

Template:

- **File:** `lottie/animations/<name>.json`
- **Creator:** `<creator name>`
- **Source:** `<URL>`
- **License:** `<license name and URL>`
- **Attribution required:** `<yes/no>`
- **Attribution text (if required):** `<text>`
