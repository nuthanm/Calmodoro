# Promotional video (optional)

Chrome Web Store accepts an optional **YouTube video URL**. Video is **not required** for publication.

## Specs

| Field | Recommendation |
|-------|----------------|
| Length | 45–90 seconds |
| Resolution | 1080p (1920×1080) |
| Format | Upload to YouTube (public or unlisted) |
| Audio | Optional soft background music (royalty-free only) |
| Content | Screen recording of actual extension — no misleading UI |

## What to show

1. Pin Calmodoro icon (3 sec)
2. Open popup → Start Focus (5 sec)
3. Close popup → highlight **badge counting down** (10 sec)
4. Session ends → break triptych appears (15 sec)
5. Accept break → countdown runs (10 sec)
6. Quick glimpse of Settings schedule + Stats (10 sec)
7. End card: "Calmodoro — Calm Pomodoro for Chrome" (5 sec)

Full script: [script-outline.md](./script-outline.md)

## Tools

- **OBS Studio** (free) — screen recording
- **Clipchamp / DaVinci Resolve** (free) — trim and titles
- **YouTube** — hosting

## Copyright

- Record only **your extension UI** and generic browser chrome
- Do not use copyrighted music, stock video, or third-party logos
- Royalty-free music: YouTube Audio Library, Pixabay Audio (check license)

## After recording

1. Upload to YouTube as **Unlisted**
2. Paste URL in Developer Dashboard → Store listing → Promotional video
3. Set a clear thumbnail (1280×720) showing Calmodoro popup or break screen
