Privacy Policy

Product: Calmodoro Chrome Extension · Effective: July 2026 · Version: 1.0.0

Summary

Calmodoro stores timer and settings data only on your device using Chrome's storage API. We do not collect, transmit, sell, or share your personal data with any server or third party.

Who this applies to

This policy describes how Calmodoro (“the Extension”) handles information when you install and use it.

Data stored locally

DataPurposeStored whereSent to our servers?
Timer state (end time, mode, paused/running)Run Pomodoro sessionschrome.storage.localNo
Settings (durations, schedule, DND, layouts)Your preferenceschrome.storage.localNo
Daily stats (sessions, skips, reminders)Progress trackingchrome.storage.localNo

This data remains on your computer inside Chrome's extension storage. Uninstalling the extension or clearing extension data removes it.

What we do not collect

  • Browsing history or page content
  • Passwords or form data
  • Location
  • Name, email, or account identifiers
  • Analytics or usage telemetry
  • Financial information

Network access

Calmodoro does not send timer or settings data to external servers.

The extension may load the DM Sans font from Google Fonts for UI display. That request is subject to Google's privacy policy. No other third-party network requests are made by the extension code.

Permissions

Permissions are used only for extension functionality:

  • storage — save settings and timer state locally
  • alarms — session end and reminder timing
  • notifications — optional break/reminder alerts
  • windows — break and reminder overlay windows
  • offscreen — accurate toolbar icon countdown

Children's privacy

Calmodoro is not directed at children under 13 and does not knowingly collect information from children.

Changes

We may update this policy when the extension changes. The effective date at the top will be revised. For Chrome Web Store installs, the policy URL in the store listing should point to the latest version hosted on this site.

Open source

Calmodoro is open source under the MIT License. You can review the source code in the GitHub repository to verify this policy.
