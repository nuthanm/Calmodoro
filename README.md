# Calmodoro

Calmodoro is a Chrome extension that combines a focused Pomodoro timer with calming animations and visual break reminders. It is designed to help users stay productive while building healthier work and rest habits throughout the day.

## Features

- Customizable focus, short break, and long break intervals
- Animated break reminders for a more calming transition between sessions
- Do Not Disturb mode to silence reminders when needed
- Simple browser-based workflow with quick access from Chrome
- Flexible routine support for study, deep work, and daily task management

## Getting Started

### Install the extension

If you have a local Calmodoro build or release folder:

1. Download the latest build files.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode**.
4. Select **Load unpacked**.
5. Choose the Calmodoro project folder.

### Development setup

To work on the extension locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/nuthanm/Calmodoro.git
   cd Calmodoro
   ```
2. Open `chrome://extensions` in Chrome.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the local project directory.
5. Make your changes in the repository and use **Reload** on the extension card to test updates.

## Usage

### Start a focus session

1. Open the Calmodoro extension from the Chrome toolbar.
2. Set your preferred focus duration.
3. Start the timer and work until the session ends.

### Take guided breaks

When a focus session finishes, Calmodoro shows an animated reminder to step away briefly. Users can then begin a short or long break before returning to the next session.

### Adjust your routine

Use the extension settings to:

- Change focus and break lengths
- Enable or disable Do Not Disturb mode
- Tune the experience to match your work style
